# officialm-aiyouxi-resource

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面。这些页面以静态资源的形式存放，不针对任何特定域名或网站，仅作为通用页面存档与分发用途。

## 目录说明

```
├── pages/           # 存放各独立 HTML 页面
├── assets/          # 页面引用的样式、脚本、图片等公共资源
├── docs/            # 项目相关文档或说明
└── README.md        # 本文件
```

- `pages/`：每个子目录或文件对应一个独立的 HTML 页面。
- `assets/`：存放页面所需的静态文件，按类型或用途组织。
- `docs/`：可选的补充文档，如页面使用说明或开发记录。

## 页面归档说明

- 每个 HTML 页面均为独立完整的内容，不依赖外部特定服务。
- 页面可包含基础的交互功能或展示内容，但均为通用设计。
- 归档页面可能不定期更新或新增，请以仓库最新提交为准。

- 欢迎通过 Issues 或 Pull Requests 提交建议，但不保证所有建议都会被采纳。

## 使用说明

1. 克隆或下载本仓库。
2. 直接在浏览器中打开 `pages/` 下的 HTML 文件。
3. 如需部署，请确保 `assets/` 目录与页面文件保持相对路径正确。
